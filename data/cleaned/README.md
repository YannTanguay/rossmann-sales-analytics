## Cleaned Dataset

A stratified random sample of approximately 6,000 observations from the cleaned Rossmann dataset has been included in this repository (2,000 rows from each year: 2013, 2014, and 2015).

This sample is provided only for demonstration purposes; all SQL queries, excel analyses, and modelling results found in this project were conducted using the complete cleaned dataset.

The full cleaned dataset has not been included due to file size constraints but may be provided upon request.
